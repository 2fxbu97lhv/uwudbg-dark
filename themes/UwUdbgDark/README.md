# UwUdbg Dark

A dark variant of [herrcore's](https://github.com/herrcore) legendary [uwudbg](https://github.com/herrcore/uwudbg) theme for x64dbg — the same kawaii chaos, now easier on the eyes. The original ships a pastel assault so aggressive you'll fix bugs at record speed just to close the window.

## Install
- Copy `themes/UwUdbgDark` into your x64dbg `themes` directory
- Open x64dbg and choose **Options → Themes → UwUdbgDark**
- (Optional) Copy `translations/` to x64dbg and switch to **Options → Languages → [af] Afrikaans** for full kawaii localization
