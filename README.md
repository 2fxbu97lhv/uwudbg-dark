# The best theme for x64dbg!

Originally created by [herrcore](https://github.com/herrcore), the x64dbg theme of all time. This repo is a dark variant, **UwUdbg Dark**: same kawaii chaos, less eye-searing pastels.

## I want it!
- Clone repository
- Copy `/themes` to x64dbg directory
- Copy `/translations` to x64dbg directory
- Open x64dbg and choose Options->Themes->UwUdbgDark
- What have you done?!
- Choose Options->Languages->[af] Afrikaans and restart debugger
- Weep as your debugger now speaks kawaii!
